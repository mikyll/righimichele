### Setup Progetto

Eclipse:
1. import progetto
2. impostare la versione del JDK del progetto:
	1. Properties (Click destro sul progetto)
	2. Build Path
	3. Libraries
	4. JRE System Library [...] > Edit
	5. Execution environment: JavaSE-1.8 (jdk1.8.0_281)
		*foto*
3. aggiungere una Run Configuration valida:
	1. Click destro > Run As... > Run Configurations... > Java Application > New Configuration
	2. Modificare il nome
	3. Selezionare la classe del main, specificando anche il package (NB: usare Search per risparmiare tempo e insulti)
	4. Runna e goditi il risultato
	5. Se si rompe Eclipse o non va qualcosa, c'Ã¨ una soluzione fantastica e che funziona sempre: usa IntelliJ
